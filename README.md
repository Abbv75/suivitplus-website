#  Chauffy — Landing Page

##  Description

Chauffy est une application malienne qui permet à un utilisateur de commander un chauffeur professionnel pour conduire **sa propre voiture** lorsqu’il est dans l’incapacité de le faire (fatigue, soirée, travail, etc.).

Ce projet est une **landing page moderne et animée** présentant le concept, les fonctionnalités et les avantages de Chauffy.

---

##  Stack technique

* React (TypeScript)
* Tailwind CSS
* shadcn/ui
* Lucide React
* GSAP (animations)
* Framer Motion

---

##  Lancer le projet

```bash
# Installer les dépendances
npm install

# Lancer en développement
npm run dev
```

---

##  Build production

```bash
npm run build
```
